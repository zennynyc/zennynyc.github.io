Summary: Lattitude vs X

The purpose of this project was to analyze how weather changes as you get closer to the equator. We accomplished this analysis by pulling dataset of 500 cities from the OpenWweatherAPI.

After cleaning up the Dataset, we used Matplotlib to plot four various aspects of the weather type vs lattitude. The types of weather includes, Maximum Temperature, cloudiness, wind speed and humidity. This site provides the sources data and Visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
